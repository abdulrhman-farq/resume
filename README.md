# Resume - Abdulrahman Al-Nuqaidan

A modern, responsive resume website built with React and Tailwind CSS.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (Icons)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is configured for Vercel deployment. Simply connect your GitHub repository to Vercel for automatic deployments.
