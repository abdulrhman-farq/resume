# Assets Folder

Please add the following image files to this folder:

1. **profile-photo.jpg** - Your professional headshot photo
2. **hudhud-logo.png** - HudHud Maps company logo
3. **balady-logo.png** - Balady company logo
4. **urbi-logo.png** - Urbi company logo
5. **google-maps-logo.png** - Google Maps company logo

The images will be automatically displayed in the resume. If an image is missing, the component will gracefully fall back to the default icon.
